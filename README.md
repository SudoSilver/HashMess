#HashMess

The dynamic Rust HashMap can store strings functions etc.
Its intended for prototyping and projects where you want to have a dynamic HashMap.

Examples to use it are in the examples directory.

##Features

-Key value pairs 
-Values of any type
-String keys
-*Panics* on missing keys
-Dynamic behavior
-Great for prototyping and projects where you want ease of development instead of type safety

##instalation 

cargo add HashMess
